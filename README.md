# lights-cameras-demo
An interactive demo of [Tangram](http://github.com/tangrams/tangram)'s light and camera types.

Live demo: http://tangrams.github.io/lights-cameras-demo/

![tangram-demo screenshot](https://cloud.githubusercontent.com/assets/459970/6629470/c007b1de-c8e4-11e4-854e-fd69f5135b3a.png)

### To run locally:

Download this repo, then start a web server in its directory:

    python -m SimpleHTTPServer 8000
    
If that doesn't work, try:

    python -m http.server 8000
    
Then navigate to: [http://localhost:8000](http://localhost:8000)
